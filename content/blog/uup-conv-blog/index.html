reating an ISO with UUP Converter

Published: 8th of March 2026

Learn what the UUP platform is, how files are assembled, and how I fully automate Windows ISO creation using my optimal configuration. These settings and modifications apply to Windows 10 and Windows 11 images.

What is UUP?

UUP (Unified Update Platform) is a technology introduced by Microsoft to make Windows updates much more efficient. Instead of distributing massive, complete ISO files for every minor update, UUP allows downloading only the strictly necessary packages (including Insider testing builds).

How are UUP files processed?

Once the packages are downloaded, you will end up with a folder full of .cab and .esd archives. This is where UUP Converter comes into play. The converter is a complex script that automates powerful command-line tools (like DISM and wimlib). It unarchives the components, merges them, integrates cumulative updates, and rebuilds everything into a bootable ISO ready to be written to a USB drive.

My Configuration (ConvertConfig.ini)

To avoid manually confirming every step, the script relies on the configuration file. Here is what the settings I use mean:

  • AutoStart = 1 & AutoExit = 1: Full automation. The script starts and closes itself automatically upon completion.
  • AddUpdates = 1: Instantly integrates the latest updates (Security patches) directly into the ISO image, saving precious time on the first system boot.
  • Cleanup = 1 & ResetBase = 2: Deletes temporary files and old update backups (ResetBase), resulting in a much cleaner and smaller image.
  • NetFx3 = 1: Injects support for .NET Framework 3.5, required for running older games and applications. Note: Including .NET 3.5 via this method is no longer valid for Windows 11 build 28000 and newer.
  • wim2esd = 1: Converts the installer file to ESD format (highly compressed), ideal for FAT32 installation partitions which have a strict 4GB per-file limit.
  • SkipEdge = 1: Skips forcing the installation of the Microsoft Edge browser during the assembly process.
  • AddDrivers = 1 & Drv_Source = \Drivers: This is a lifesaver for modern PCs. It automatically fetches and integrates any extracted driver (like .inf/.sys files) that you place in the \Drivers folder next to the script. This is essential if you have a modern Intel processor with VMD/RST technology enabled; without injecting these storage drivers into the ISO, the Windows installer simply will not see your SSD.
  • StubAppsFull = 1 & CustomList = 1 ([Store_Apps] section): This is my main method for achieving a completely bloat-free Windows. By enabling CustomList, the script ignores useless pre-installed apps (like TikTok, games, ads) and installs only the applications I specify in a separate file named CustomAppsList.txt.

ConvertConfig.ini Code

[convert-UUP]
AutoStart    =1
AddUpdates   =1
Cleanup      =1
ResetBase    =2
NetFx3       =1
StartVirtual =0
wim2esd      =1
wim2swm      =0
SkipISO      =0
SkipWinRE    =0
LCUwinre     =0
LCUmsuExpand =1
UpdtBootFiles=1
ForceDism    =1
RefESD       =0
SkipLCUmsu   =0
SkipEdge     =1
AutoExit     =1
DisableUpdatingUpgrade=1
AddDrivers   =1
Drv_Source   =\Drivers

[Store_Apps]
StubAppsFull =1
CustomList   =1

CustomAppsList.txt File (Bloat-Free System)

To keep the system as fast and clean as possible, I use a strict list of core applications. Everything not found on this list is completely removed from the installation. Here is why I kept these specific packages:

  • Core System & Security: I kept WindowsStore, StorePurchaseApp, and DesktopAppInstaller so I can still securely download and manage software. SecHealthUI is retained to ensure the Windows Defender interface remains accessible.
  • Essential Utilities: WindowsCalculator, Paint, ScreenSketch (Snipping Tool), WindowsCamera, and WindowsSoundRecorder are lightweight, indispensable tools for daily tasks that don't slow down the system.
  • Media & Codecs: Windows.Photos and ZuneMusic (Media Player) are kept for basic media viewing. The rest of the list (HEIF, HEVC, VP9, Webp, DolbyAudio, AV1, etc.) consists of crucial video and audio codecs. Including these ensures that all images and video formats play natively out-of-the-box without needing to immediately download third-party software like VLC.
